## Run the Task Manager App Locally
These steps are not necessary until and unless the Github Pages deployment fails. The app can readily be accessed from the gh-pages link mentioned in the Task Manager App section above.

1. Clone this repository
2. Ensure that you have node and npm installed on your system
3. From the root of the project directory, run: <br/>
```npm start```
4. To now view the app, open your browser and enter: <br/>
```http://localhost:3000/task-manager-frontend```
